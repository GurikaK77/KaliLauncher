Put the finished Zombie Apocalypse modpack files here.
Supported folders/files include: mods, config, defaultconfigs, resourcepacks, shaderpacks, saves, options.txt, optionsof.txt, optionsshaders.txt, manifest.json and modlist.html.
When the user presses Download in Gurika Resources, this whole folder is copied into KaliLauncher userData/.kalilauncher/modpacks/zombie-apocalypse.
